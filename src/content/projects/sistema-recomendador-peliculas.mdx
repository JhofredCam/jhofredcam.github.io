## Overview

Este proyecto consiste en un sistema de recomendación de películas implementado como una API RESTful con FastAPI. Utiliza algoritmos de Machine Learning, específicamente la similitud del coseno, para sugerir películas a los usuarios. El sistema incluye procesos de Extracción, Transformación y Carga (ETL) para preparar los datos y está desplegado en la plataforma Render.

## Características

*   **API RESTful:** Desarrollada con FastAPI para ofrecer endpoints eficientes y bien documentados.
*   **Recomendación por Similitud del Coseno:** Algoritmo principal para identificar películas similares.
*   **Procesos ETL:** Manejo de datos desde diversas fuentes, su transformación y carga para el modelo de recomendación.
*   **Despliegue en Render:** La aplicación está disponible y operativa en la nube.

## Tecnologías Utilizadas

*   **Python:** Lenguaje principal de programación.
*   **FastAPI:** Framework moderno y rápido para construir APIs.
*   **scikit-learn:** Para cálculos de similitud del coseno y otras operaciones de ML.
*   **pandas:** Para manipulación y análisis de datos durante el proceso ETL.
*   **Render:** Plataforma de despliegue en la nube.

## Implementación

El proyecto abarca desde la recolección y preprocesamiento de datos de películas hasta el entrenamiento del modelo de recomendación y la exposición de las funcionalidades a través de la API. Se puso especial énfasis en la eficiencia del modelo y la robustez del despliegue.
