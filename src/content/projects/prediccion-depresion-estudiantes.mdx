## Overview

Este proyecto implementa un modelo de Machine Learning para predecir el riesgo de depresión en estudiantes universitarios. El objetivo principal es identificar factores clave relacionados con el rendimiento académico y las características demográficas que pueden influir en la salud mental de los estudiantes, permitiendo una intervención temprana.

## Tecnologías Utilizadas

*   **Python:** Lenguaje principal de programación.
*   **scikit-learn:** Biblioteca para la construcción y evaluación de modelos de Machine Learning.
*   **pandas:** Utilizado para la manipulación y análisis de datos.

## Metodología

Se realizó un preprocesamiento exhaustivo de datos que incluyó la limpieza, transformación y selección de características relevantes. Posteriormente, se entrenaron y evaluaron diversos algoritmos de clasificación de Machine Learning para determinar el modelo más adecuado para la predicción.

## Resultados

El modelo desarrollado mostró una capacidad prometedora para identificar a estudiantes con alto riesgo de depresión, lo que podría servir como una herramienta de apoyo para las instituciones educativas en la implementación de programas de bienestar estudiantil.
